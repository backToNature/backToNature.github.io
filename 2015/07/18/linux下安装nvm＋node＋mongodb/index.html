linux下安装nvm＋node＋mongodb

linux&mac os卸载nodejs

安装nvm（管理nodejs版本的工具）

  1. 下载nvm:

    $ git clone git@github.com:creationix/nvm.git ~

  2. 配置终端启动时自动执行 source ~/nvm/nvm.sh,在~/.bashrc, ~/.bash_profile, ~/.profile或者~/.zshrc文件添加如下内容

    source ~/nvm/nvm.sh

  3. 重新打开终端，输入nvm可以看到命令提示
